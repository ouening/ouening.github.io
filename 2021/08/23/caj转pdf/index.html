caj转pdf

  • 2021-08-23
  • Wenqing Zhou

caj文件是知网自己开发的一种文档格式，衡多时候并不方便阅读和做笔记，因此在网上找到了caj2pdf这个工具，项目地址在：
https://github.com/caj2pdf/caj2pdf

1.环境依赖：

  • python3.3+
  • pypdf2 (pip install -U pypdf2)
  • mutool (windows下需要从https://mupdf.com/index.html下载，并放到`caj2pdf.py`相同目录下)
